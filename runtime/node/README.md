# runtime/node/

Portable Node.js v22 runtime (~94 MB extracted).

Populated by `build.cmd` — downloads the official Windows x64 zip from [nodejs.org](https://nodejs.org/dist/v22.16.0/node-v22.16.0-win-x64.zip) and extracts it here.

Key files after build:
- `node.exe` — Node.js executable
- `npm.cmd` — npm batch launcher
- `npx.cmd` — npx batch launcher
